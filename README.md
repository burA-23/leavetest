##Getting Started 

###Pre-requisites 

This project requires the following dependencies: 

**Programming Languages**: Python, HTML 
**Package Manager**: Pip 


###Installation 
Build leavetest from source and install dependencies: 

**1. Clone the repository:** 

	> *git clone https://github.com/burA-23/leavetest* 

**2. Navigate to the project directory:**

	> *cd leavetest* 

**3. Install the dependencies:**

Using **Pip**: 

	> *pip install -r requirements.txt* 

**Usage** 

**Run the project with:**  

Using **pip**: 

	> *python {entrypoint}*

**Testing** 

Leavetest uses {test_framework} test framework. Run the test suite with:
Using **pip** 

	>pytest

Using pip 

	>pytest
